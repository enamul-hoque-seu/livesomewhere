# Live Somewhere

**Where people and progress live together**

Live Somewhere is a Vite + React + TypeScript platform focused on community learning, progress tracking, and modern content experiences.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start local development:
   ```bash
   npm run dev
   ```
3. Run tests:
   ```bash
   npm run test
   ```
